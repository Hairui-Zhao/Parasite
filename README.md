# Parasite:面向低排队延迟目标的作业调度框架
**对应文章第6章**

在深度学习研发集群中，**探索性作业**（如超参数调优）占比超过99%，但其短时、高频的特性与现有面向长时确定性作业的调度器严重不匹配，导致探索性作业排队延迟过长，严重阻碍研发效率。

为解决这一矛盾，我们提出了 **Parasite** 框架。其核心思想是让探索性作业以低干扰方式**寄生**于已占用的GPU资源上，与确定性作业安全共享计算设备，从而大幅削减探索性作业的排队与执行时间。

框架通过两大核心技术实现高效安全的资源共享：首先，基于**内存碰撞概率模型**与多维度性能建模，构建了精准的干扰预测与放置优化算法，能在庞大搜索空间中为探索性作业找到最优放置位置。其次，**作业融合机制**在内核层级进行调度，通过动态编排不同作业内核的执行顺序，主动缓解资源竞争，最小化性能干扰。
